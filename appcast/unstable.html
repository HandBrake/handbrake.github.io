HandBrake 1.5.1

The HandBrake Team is pleased to announce the release of HandBrake 1.5.1

Upgrade Notice

Before updating, please make sure there are no pending encodes in the Queue. If you are currently using HandBrake older than version 1.4.0, please also make note of any custom presets you have created, as they may not be compatible with newer versions.

View the full release notes on GitHub.

System Requirements

  • macOS 10.13 High Sierra or later
